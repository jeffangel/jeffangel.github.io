Train a CNN model from scratch using Keras API

En este post, te mostraré uno de los flujos a seguir para entrenar un modelo CNN desde cero a fin de estimar la edad de una persona y predecir su sexo a partir de imágenes faciales.
Como fuente de datos se utiliza un subconjunto de datos de imágenes de rostros extraídos del dataset de Kaggle UTKFace, sin embargo, el enfoque es aplicable a cualquier conjunto de datos de imágenes.
Para la creación de esta red neuronal convolucional utilizaremos la API de Keras v3, ahora integrado a Tensorflow a partir de la versión 2.0.
Considero importante mencionar que Keras v3 ofrece 3 formas de crear modelos, puedes consultar el detalle completo en el sitio web de Keras. De igual forma, te comento brevemente la diferencia entre estas:

  1. Secuencial: Forma simple de apilar capas, soporta una sola entrada y salida.
  2. API funcional: Permite arquitecturas más personalizables con múltiples entradas y salidas.
  3. Sub-clase de modelo: Permite crear capas personalizadas para proyectos complejos.

Debido a que la arquitectura a crear generará dos predicciones de salida (edad y genero), utilizaremos la API funcional. Puedes seguir el notebook disponible en Google Colab.
